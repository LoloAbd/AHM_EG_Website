About

This project offers several key functions to assist elderly individuals:


  • Setting Time and Date: Users can set the current time and date by pressing the (*) button.
  • Medicine Reminder: Three alarms (A, B, C) can be set using the keypad to remind users about medication times. A buzzer sounds and a message is displayed on the LCD to remind the user to take their medication.
  • Heart Rate Check: Users can measure their heart rate by placing their finger on the sensor.
  • Temperature Check: The temperature can also be measured using the same sensor.
  • Fainting Detection: The system can detect if the user has fainted.
  • Medication Confirmation: The buzzer continues to sound until the user presses the (D) button to confirm they have taken their medication.
  • Communication with Caregiver: The system sends messages to the caregiver when itâ€™s time for medication and when the user confirms medication intake.
  • Fall Detection: If the system detects a fall, it immediately sends a message to the caregivers through WhatsApp.
